# Interview

## When to use

Use when the user needs guided requirements discovery before generating a seed.

## How to invoke

Run `mobius interview` via the Bash tool. For example:

```text
Bash('mobius interview')
```

Preserve any user-provided arguments and pass them to `mobius interview` using normal shell quoting.
