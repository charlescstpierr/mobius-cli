# Setup

## When to use

Use when the user wants to install, inspect, or remove Mobius integrations.

## How to invoke

Run `mobius setup --runtime claude` via the Bash tool. For example:

```text
Bash('mobius setup --runtime claude')
```

Preserve any user-provided arguments and pass them to `mobius setup` using normal shell quoting.
