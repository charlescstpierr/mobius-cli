# Help

## When to use

Use when the user asks what Mobius can do or needs command usage.

## How to invoke

Run `mobius --help` via the Bash tool. For example:

```text
Bash('mobius --help')
```

Preserve any user-provided arguments and pass them to `mobius help` using normal shell quoting.
